贴墙

  • 问题的突破口就在于小瓷砖的面积大小，当墙的面积是2的倍数时，则无法将墙壁都贴满，反之一定能够贴满

    • 如果先将墙的面积算出来的话就需要用到long long的数据类型来储存给的值
    • 用int应该也是可以的，就这题而言，相乘溢出是不会改变结果的奇偶性的
    • 代码如下
    #include <iostream>

using namespace std;

int main() {
    long long h, w;
    cin >> h >> w;
    if (h * w % 2 == 0) { // 如果是2的倍数
        cout << "Yes" << endl;
    } else {              //如果不是2的倍数
        cout << "No" << endl;
    }
    return 0;
}
    • 还有一种更好的办法，不难发现两个正整数相乘结果是奇数的充分必要条件是两个正整数都是奇数，固只需要判断两个数是否同时为奇数（即只要有一个偶数就输出Yes）便可（这是本题真正想考的思路）
    • 代码如下
    #include <iostream>

using namespace std;

int main() {
    int h, w;
    cin >> h >> w;
    if (h % 2 == 0 || w % 2 == 0) {//如果其中有一个是偶数
        cout << "Yes" << endl;
    } else {
        cout << "No" << endl;
    }
    return 0;
}

雕像

  • 一道简单的模拟题，需要注意的一个坑就是如果是闰年的2月29日制作的雕像的话，纪念日就要等到闰年才能庆祝一次
  • 代码如下
#include <iostream>

using namespace std;

int main() {
    int y1, m1, d1, x, y2, m2, d2;
    cin >> y1 >> m1 >> d1 >> x >> y2 >> m2 >> d2;
    int ans;
    if (m1 == 2 && d1 == 29) {//如果是闰年的2月29号
        ans = x;
        for (int i = y1 + 1; i <= y2; i++) {//从第二年开始算,算到今年
            if (i % 400 == 0 || (i % 4 == 0 && i % 100 != 0)) {//如果是闰年
                ans++;//答案加一
            }
        }
        if (y2 % 400 == 0 || (y2 % 4 == 0 && y2 % 100 != 0)) {//如果今年是闰年
            if (m2 == 1) {//今年的纪念日还没到
                ans--;//答案减一
            }
            if (m2 == 2 && d2 < 29) {//今年的纪念日还没到
                ans--;
            }
        }
    } else {//如果不是，则每年都会有一次纪念日
        ans = x + y2 - y1;
        if (m2 < m1) {//如果今年的纪念日还没到
            ans--;
        }
        if (m2 == m1 && d2 < d1) {//如果今年的纪念日还没到
            ans--;
        }
    }
    cout << ans << endl;
    return 0;
}

关于n!的问题

先上题目：

链接：https://ac.nowcoder.com/acm/contest/9692/D 来源：牛客网

给定数n，求n!的p进制下有多少个后导零。非常简单。

为了简化问题，p保证为素数。

输入描述:

第一行给定一个数t，表示有t组输入
接下来t行，每行给定两个数n,p；意义如题所示；
输入范围：(t<=1000)  (1<=n<=1000000 ） (2<=p<=1000000)

输出描述:

对于每一组输入，输出单独的一行表示答案。

示例1

输入

复制

2
2 2
3 2

输出

复制

1
1

这是一场新生赛的题目，我花了好久时间一直在搞那个进制转换的模板，但是结果是我WA了5发。比完赛后补题的时候才发现原来这是一个简单的数学问题。

题中所说:“求n!的p进制下有多少个后导零”，要产生后导零就必须用质因数的个数，比如说：求6的阶乘的3进制下有多少个后导零，6的阶乘显然是6 * 5 * 4 * 3 * 2 * 1 将其转化为6进制后有多少个零就相当于看这个1~6中质因子3的倍数的个数，而n!中质因子p的个数，实际上等于1~n中p的倍数的个数n/p +（n/p）！，于是可以看到递归的影子，递归版本如下：

int cal(int n, int p){
	if(n < p) return 0;
	return n / p + cal(n/p, p);
}

而当然亦可以用循环来代替递归：

long long res;
while(n >= p){
    res += n/p;
    n /= p;
}
